Deploy a sensor, one needs to run an executable with a specific configuration. This task is simplified by the use of the present tool that produces a boilerplate environment for a d4-client.

You only need to specify:
Whether you prefer to use the lightweight C client or the Go client
Your architecture / OS
The type of data you want to ship
The destination server

If you do not want to host your own D4-server you can choose to use crq.circl.lu

If you fill in your email address, CIRCL will send you API credentials to access our APIs (eg. passiveDNS). Every valid packet you send to crq grant you additional quota for querying our APIs!
